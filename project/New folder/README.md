# Biomedical Engineering in Breast Cancer Treatment (static site)

Open `index.html` in a browser to view the project.

Files:

- index.html — main page with sections and SVG illustrations
- styles.css — styles for layout and visual design

Multi-page site:

- `index.html` — Home/overview
- `about.html` — About this page
- `imaging.html` — Imaging details
- `therapy.html` — Targeted therapy and nanotech
- `surgery.html` — Surgical tech and robotics
- `reconstruction.html` — Tissue engineering and reconstruction
- `contact.html` — Contact details

Notes:

- Images on the page use Unsplash links (free-to-use images). Replace with local images by placing them in the folder and updating the `src` attributes.
- This site is an educational overview and not medical advice.

About:

- This project is a concise, introductory overview of how biomedical engineering applies to breast cancer treatment, intended for students and general audiences. It highlights imaging, therapy, surgery, and reconstruction technologies with illustrative images and simple schematics.

To preview locally (Windows):

1. Open the folder in File Explorer.
2. Double-click `index.html` or open with your browser.
